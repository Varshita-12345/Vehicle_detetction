# Vehicle Detection and Counting using YOLOv4

 Implements a real-time vehicle detection and counting system using **YOLOv4** and **OpenCV**.
The model detects vehicles such as cars, buses, trucks, and motorbikes from a video stream and counts them when they cross a predefined line.

This project demonstrates the use of ** computer vision techniques** for traffic analysis and intelligent transportation systems.

## Download YOLO Weights
